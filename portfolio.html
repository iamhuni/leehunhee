Portfolio Process
전체적으로 심플하면서도 사용자의 편의를 우선으로한 레이아웃으로 고려해 디자인하였고,
전반적인 내용은 메인 페이지에서 확인 할 수 있도록 했으며, 포트폴리오 프로세스 상세페이지에서는
이젠 컴퓨터 아카데미 학원에서 6개월간 배운 내용을 바탕으로 제작한
포트롤리오를 담은 제작 과정을 소개하였습니다.
Brainstorming
브레인스토밍을 통해 포트폴리오를 주제로
다양하게 그려보았습니다.
Mind Map
브레인스토밍에서 다양한 모습을 그려낸 후
마인드 맵을 통해 그룹화시켜 보았습니다.

Image Board / Idea Sketch
사용자의 편의와 사용성을 고려해 전체적인 페이지 레이아웃이 한 눈에 들어올 수 있도록 제작하였습니다.
Wireframe
와이어 프레임을 통해 전체적인 포트폴리오의 레이아웃을 감상하실 수 있습니다.

Information Architecture
전체적인 흐름을 파악 할 수 있도록 정보구조를 설계하였습니다.

Flow Chart
사용성과 서비스 구성의 정리를 위한 플로우 차트를 설계했습니다.

Style Guide
Web Site Identity
포트폴리오의 분위기를 따뜻함을 강조함으로써 브라운과 베이지 계열의 컬러를 사용했습니다.
직관적이고 편리한 레이아웃에 디자인하였고, 전반적인 내용을 메인 페이지에서 확인 할 수
있도록 디자인 하였으며, 포트폴리오 프로세스 페이지에 NCS 과정을 담아 수료과정과
포트폴리오의 전체 기획부터 제작 단계까지 자세하게 확인 하실 수 있습니다.

Logo

Meaning

제 이름 자체가 브랜드가 될 수 있도록 이름 '이훈희'의 자음인 'ㅎㅎ'을 활용하여 로고를 만들어보았습니다.
'ㅎㅎ'는 우리나라에서 많이 쓰이는 웃는 의성어입니다. 저도 로고의 의미 처럼 만족스러운 결과물을 만들어
많은 사람들에게 웃음을 줄 수 있는 웹퍼블리셔가 되고자 자음을 활용해 로고에 의미를 더해보았습니다.

창원단감아삭체를 사용하여 깔끔하면서도 귀여운 이미지를 받을 수 있도록 하였고 전체적인 포트폴리오의
분위기와 어우러지도록 따뜻한 느낌이 나는 컬러를 사용해 진행 하였습니다.

로고를 이용해 타이포그래피로 다양하게 디자인해 보았습니다.
